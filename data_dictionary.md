# Data Dictionary: Customer Shopping Behavior Dataset

| Variable | Description | Data Type |
|--------|------------|----------|
| Customer ID | Unique identifier for each customer | Categorical (Integer) |
| Age | Age of the customer at time of purchase | Numerical (Integer) |
| Gender | Self-reported gender of the customer | Categorical (String) |
| Item Purchased | Specific product purchased | Categorical (String) |
| Category | Product category of the item purchased | Categorical (String) |
| Purchase Amount (USD) | Monetary value of the transaction in USD | Numerical (Integer) |
| Location | Geographic location of the customer | Categorical (String) |
| Size | Size selected for the purchased item | Categorical (String) |
| Color | Color of the item purchased | Categorical (String) |
| Season | Season during which the purchase occurred | Categorical (String) |
| Subscription Status | Indicates whether the customer is subscribed | Categorical (Binary) |
| Shipping Type | Shipping option selected for the order | Categorical (String) |
| Discount Applied | Indicates whether a discount was applied | Categorical (Binary) |
| Promo Code Used | Indicates whether a promo code was used | Categorical (Binary) |
| Number of Previous Purchases | Count of purchases made before the current transaction | Numerical (Integer) |
| Payment Method | Payment method used for the transaction | Categorical (String) |
| Frequency of Purchases | Typical purchase cadence of the customer | Categorical (Ordinal) |
